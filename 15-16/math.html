Math of the game

The height of an equiliteral triangle

`a^2 + b^2 = c^2`

`(1/2)^2 + h^2 = 1^2`

`1/4 + h^2 = 1`

`h^2 = 1 - 1/4`

`h = sqrt(1 - 1/4)`

`h = sqrt(3/4)`

`h = sqrt(3) / sqrt(4)`

`h = sqrt(3) / 2`

`h ~~ 0.866`

The viewing angle

world space triangle height: `h_w = sqrt(3)/2`

view space triangle height: `h_v = 1/2`

`h_w * cos(alpha) = h_v`

`cos(alpha) = h_v / h_w`

`alpha = arccos(h_v / h_w)`

`alpha = arccos((1/2) / (sqrt(3)/2))`

`alpha = arccos(1/sqrt(3))`
